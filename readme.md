# A Gameboy Emulator
Author: Don Freiday

My very first emulator. Implemented so far: very little.
I've borrowed a bit of GBEPlus (CPU registers, HLE bios state, etc), am implementing opcodes, and using the excellent BGB debugger to verify my work.
